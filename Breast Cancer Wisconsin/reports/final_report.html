Diagnostic support · WDBC · Tier A

A handful of nuclear measurements tell malignant from benign almost perfectly — so the real question is the operating point, not the algorithm

A diagnostic + predictive analysis of 569 fine-needle-aspirate samples (357 benign / 212 malignant), built to the McKinsey communication standard.

Situation An FNA yields 30 nuclear-morphology numbers per breast mass; a pathologist must call each mass malignant or benign, and a missed malignancy is the dangerous error.

Complication It is unclear whether all 30 measurements are needed, which carry the signal, and whether a classifier is dependable enough — at a high-sensitivity threshold — to serve as a second reader rather than a curiosity.

Resolution It is, with caveats.

Nuclear morphology separates malignant from benign nearly perfectly — a regularized linear model reaches 0.996 cross-validated ROC-AUC — but the signal is concentrated in a few size and irregularity “worst-case” measurements that are heavily redundant; so the deployment decision is which operating point to run, and — with no demographic data to audit — the model is defensible only as a second reader.
  1. The signal is concentrated, not spread. The top separators are all size/irregularity features (concave points_worst (d=2.7), perimeter_worst (d=2.6), concave points_mean (d=2.5), radius_worst (d=2.5)); a single feature (perimeter_worst) already reaches AUC 0.976, while texture and standard-error features sit near zero effect.
  2. The 30 features are only ~3 real signals. Collinearity is extreme (23/30 features with VIF>10, max 3806; 21 feature pairs at |r|>0.9), and 63% of variance sits in the first two principal components — so no single measurement is irreplaceable and regularization is mandatory.
  3. Every model clears ROC-AUC ≈ 0.98 — the choice is the threshold. At a sensitivity-first operating point the winner catches 41 of 42 malignancies (97.6% sensitivity) for 4 false alarms (94.4% specificity) on the held-out test.
Recommendation. The clinical-decision-support lead should pilot the L2-logistic classifier as a second-reader flag at threshold ≈ 0.15 (tuned for ≥98.0% sensitivity), prioritize capturing the “worst” size/irregularity features, and — because the data carries no demographics to audit — commission prospective multi-site validation before any autonomous role.

So What: All five model families land within a whisker of each other at ROC-AUC ≈ 0.98 and PR-AUC ≈ 0.99. When the algorithm stops being the differentiator, the decision moves to the operating point — which is a clinical judgement about the cost of a false negative, not a modelling one.

Part I · Key Line 1 — Diagnostic

Malignant nuclei separate on size and irregularity — texture barely matters

Splitting each feature by diagnosis and ranking by Cohen's d (standardized mean difference), the separation is dominated by size (radius/perimeter/area) and border irregularity (concavity/concave-points), and disproportionately by the worst summary — the most extreme nuclei in the image. 26 of 30 features survive Benjamini-Hochberg FDR correction, but effect size, not the p-value, is what matters here.

So What: Feature-capture effort and clinical attention should concentrate on the size/irregularity “worst” measurements. The weakest features (symmetry_se, texture_se, fractal_dimension_mean, smoothness_se) carry almost no class signal and can be dropped without loss.
So What: One feature alone (perimeter_worst) ranks malignant above benign at AUC 0.976 — near-clinical performance from a single number. This is the honest bar the full model must clear, and it does: 0.996 vs 0.976 CV AUC.

Part II · Key Line 2 — Structure

Thirty columns, but only a few independent signals

The features are redundant by construction (perimeter ≈ 2πr, area ≈ πr²; concavity, concave-points and compactness co-move). The correlation structure and the VIF confirm it, and PCA compresses 73% of all variance into three components.

So What: 21 feature pairs exceed |r|>0.9 and 23/30 features have VIF>10 (max 3806). Feeding all 30 raw features into a linear or distance model double-counts the same signal — regularization or PCA is not optional, and per-feature importances will be unstable.
So What: PC1 alone (~44% of variance) nearly splits the two classes. The effective dimensionality is a handful, not thirty — which is exactly why a simple regularized model wins and a complex one buys nothing.

Appendix · Methodology & caveats

Why no single feature “wins” the model — and why that's fine

Permutation importance on the winner is tiny and dispersed (largest single-feature drop = 0.0098 AUC): removing any one size/irregularity feature barely hurts because its collinear siblings substitute. This is the model-side echo of the VIF result, not a contradiction of Key Line 1 — the trustworthy driver ranking is the univariate Cohen's d.

Method

  • Data: WDBC, 569 FNA samples, 30 nuclear-morphology features (10 measurements × mean/SE/worst). One row per sample; SHA-256 lineage logged; a phantom all-null trailing column dropped at ingestion.
  • Path: A (diagnostic) → B (predictive classification). Path C (causal) is deliberately not run — morphology is an observed marker, not an intervention; no counterfactual is identifiable.
  • Target & metric: Malignant = 1 (positive by cost). Led by ROC-AUC, PR-AUC and malignant recall; accuracy reported but never used for selection.
  • Validation: RepeatedStratifiedKFold (10×3) for model selection; a stratified 20% test set (114 samples) scored once. Seed 42. The winner is well calibrated (Brier 0.0209).
  • Leakage discipline: scaling and PCA fit inside the Pipeline on train folds only; the operating threshold chosen on training-fold CV predictions, never the test set.

Fairness & limitations

The fairness audit cannot be run. No demographic attributes exist in WDBC (age/race/site absent), so subgroup performance cannot be audited. Deployment claim capped at second-reader use pending prospective multi-site validation.

  • Single-source historical sample (n=569, Wisconsin); no external validation cohort.
  • No demographics -> fairness subgroup audit impossible (see above).
  • 'Worst' features may be instrument/segmentation-specific; portability unproven.
  • Diagnostic/association only -> decision-support, never autonomous diagnosis.
  • Near-perfect separability is a known property of this benchmark; real-world screens are harder.
